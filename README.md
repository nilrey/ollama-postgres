    Технические средства: RAM 32Gb, HDD 1Tb, CPU i3-13100 (3.40 GHz)
    Операционная среда: Windows 11
    IDE разработки: VSCode 
    Язык раработки: Python 3.11.9
    Установлено: Ollama 0.11.11.0, Postgresql 12
